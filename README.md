# ORP-Data-Logger-Web
Visualiseur Red Ox Potentiel pour piscine.
Inclu:
  - enregistrement des données sur cart SD
  - RTC
  - Afficher OLED
  - Avertissement Leds (err. & enregistrement)
  - Web Access
  - Graphique (javascript)

Matériel:
  - ESP32
  - OLED 0,96"
  - RTC module
  - SD carte module
  - 2 led
  - Alim 5v
